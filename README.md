# Exemplo de persistencia de dados
Este repositório possui dois programas em C com a seguinte estrutura
```c
struct Cliente {
    char nome[50];
    int idade;
    char email[100];
};
```
- O primeiro clientes.c cria um arquivo de texto chamado clientes.csv com 5 clientes cadastrados
- O segundo lê o arquivo clientes.csv e faz um relatório de quantos clientes e calcula a média de idade
## Tecnologias
- Dev C++

## Passos para testar
Abra os arquivos .c com Dev C++ e execute.
